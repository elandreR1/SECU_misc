UE SECU - Développement d'une messagerie instantanée sécurisée

Votre application

Pour cette application vous avez accès aux pages suivantes:
  • Votre messagerie instantanée (dans le répertoire etu) avec l'identité d'Alice.
  • Votre messagerie instantanée (dans le répertoire etu) avec l'identité de Bob.
  • Votre messagerie instantanée (dans le répertoire etu) avec l'identité de Charly.

  • L'outil de "filtrage réseau" (pour l'application du répertoire etu) qui permet d'accèder à tous les messages échangés.
  • Une interface "intrus" qui étend l'interface précédente avec des fonctionnalité supplémentaires.

La messagerie que vous aviez attaquée

Pour cette ancienne version, vous avez également accès aux pages suivantes:
  • La messagerie instantanée que vous aviez attaquée. Avec l'identité d'Alice.
  • La messagerie instantanée que vous aviez attaquée. Avec l'identité de Bob.
  • La messagerie instantanée que vous aviez attaquée. Avec l'identité de Charly.

  • L'outil de "filtrage réseau" pour l'ancienne messagerie.
  • Une interface "intrus" pour l'ancienne messagerie.

  • Une calculette cryptographique qui propose toutes les opérations cryptographiques de base.
